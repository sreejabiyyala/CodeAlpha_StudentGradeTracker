# Student Grade Tracker

## Description
This project is a Java console-based Student Grade Tracker developed for the CodeAlpha Java Programming Internship. It allows users to add student details, view student records, calculate average marks, and find the highest and lowest scores.

## Features
- Add Student
- View Students
- Calculate Average Marks
- Find Highest Marks
- Find Lowest Marks
- Console-Based Menu

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
